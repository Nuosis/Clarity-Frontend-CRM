# Notes Feature - API Contracts

## Backend API Endpoints

Base URL: `https://api.claritybusinesssolutions.ca`

### Authentication
All requests require HMAC-SHA256 authentication:
- Header: `Authorization: Bearer {signature}.{timestamp}`
- Generated by: `dataService.generateBackendAuthHeader()`
- See: `src/services/dataService.js`

## Endpoints

### 1. List Notes

```
GET /api/notes
```

**Query Parameters:**

**Response:**
```json

    {

      "organization_id": "uuid",
      "[TO BE DOCUMENTED]": "..."

    "limit": 0,

```

**Errors:**
- `400` - Invalid query parameters
- `401` - Authentication failed
- `403` - Unauthorized access
- `500` - Server error
