## Authentication Architecture

**Primary:** HMAC-SHA256 M2M authentication
- Secret: `VITE_SECRET_KEY` from `.env`
- Format: `Bearer {signature}.{timestamp}`
- Generated by: `dataService.generateBackendAuthHeader()`

**Secondary:** JWT-based user authentication (Supabase)
- Organization scoping: `X-Organization-ID` header
- User tracking: `created_by`, `updated_by` from JWT
- RLS policies enforce organization isolation
